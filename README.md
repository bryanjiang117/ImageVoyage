# ImageVoyage

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Testing for development
```
npm run test
```

### Compiles and minifies for production
```
npm run build
```

### To run API server (Important!)
```
cd server (To select server folder)

node server.js
```
